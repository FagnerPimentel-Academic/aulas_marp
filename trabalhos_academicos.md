# Trabalhos Acadêmicos

### Pesquisa científica na graduação e pós-graduação

Fagner Pimentel

---

# Por que fazer pesquisa?

A pesquisa acadêmica permite:

- gerar **novo conhecimento**
- resolver **problemas reais**
- desenvolver **pensamento crítico**
- avançar o **estado da arte**

---

# O que é um trabalho acadêmico?

Um trabalho acadêmico é uma **investigação sistemática** sobre um problema.

Ele utiliza:

- método científico
- revisão de literatura
- análise de resultados

---

# Tipos de trabalhos acadêmicos

- Projeto de pesquisa
- Iniciação científica
- Monografia
- Dissertação
- Tese

---

# Evolução na carreira acadêmica
Graduação: Iniciação Científica → Monografia (TCC)
↓
Mestrado: Dissertação
↓
Doutorado: Tese


Cada nível exige **maior profundidade científica**.

---

# Projeto de pesquisa

Documento que **planeja a pesquisa**.

Define:

- problema de pesquisa
- objetivos
- metodologia
- cronograma

---

# Estrutura de um projeto

| Elemento | Função |
|---|---|
| Introdução | contextualizar o tema |
| Problema | pergunta da pesquisa |
| Objetivos | o que será investigado |
| Metodologia | como a pesquisa será feita |
| Cronograma | planejamento temporal |
| Bibliografia | base científica |

---

# Iniciação científica

Primeiro contato com a pesquisa.

Objetivos:

- aprender **método científico**
- participar de projetos
- produzir relatórios e artigos

---

# Monografia

Trabalho de conclusão de **graduação**.

Características:

- análise aprofundada de um tema
- aplicação de métodos científicos
- estrutura acadêmica formal

---

# Estrutura da monografia

- Introdução
- Referencial teórico
- Metodologia
- Resultados
- Discussão
- Conclusão

---

# Dissertação

Trabalho de conclusão de **mestrado**.

Características:

- pesquisa **original**
- maior rigor metodológico
- contribuição científica

---

# Tese

Trabalho de conclusão de **doutorado**.

Características:

- pesquisa **inovadora**
- contribuição significativa para a área
- desenvolvimento de novos métodos ou teorias

---

# Comparação dos trabalhos

| Trabalho | Nível | Complexidade |
|---|---|---|
| Projeto | planejamento | baixa |
| IC | graduação | média |
| Monografia | graduação | média |
| Dissertação | mestrado | alta |
| Tese | doutorado | muito alta |

---

# Estrutura do método científico
Observação
↓
Pergunta de pesquisa
↓
Hipótese
↓
Experimentos / coleta de dados
↓
Análise
↓
Conclusão


---

# Metodologia de pesquisa

Pode incluir:

- experimentos
- simulações
- estudos de caso
- análise estatística
- modelagem computacional

---

# Resultados científicos

Uma pesquisa pode gerar:

- artigos científicos
- relatórios técnicos
- softwares
- bases de dados
- protótipos

---

# O que pode ser publicado?

- estudo de caso
- revisão de literatura
- comparação de métodos
- experimentos
- análise de dados

---

# Exemplos de contribuições científicas

- novos algoritmos
- melhorias de métodos existentes
- análise estatística de dados
- construção de datasets
- avaliações experimentais

---
